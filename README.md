A simple TypeScript project for bowling score calculation.

## Setup Instructions

### 1. **Clone the Repository**

```bash
git clone git@github.com:ojayjonathan/bowling.git
cd bowling
```

### 2. Install dependencies

Make sure you have Node.js and npm installed, then run

```bash
npm install
```

### 3. Running the Project

1. Compile Typescript

```bash
npm run compile
```

1. Execute the program

```bash
npm run exec
```

1. Run tests

```bash
npm test
```
